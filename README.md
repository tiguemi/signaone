# Signa One - Site vitrine Angular

Site vitrine pour une société de conception d'enseigne **Signa One**.

## Stack

- Angular standalone (v20) avec signaux (`signal`, `computed`).
- Bootstrap 5.3 (CDN) pour la grille et les composants UI.
- Reactive Forms pour la demande de devis.

## Fonctionnalités

- Présentation des services.
- Section "Nos concrétisations" avec filtrage par catégorie.
- Formulaire de demande de devis.

## Lancer le projet

```bash
npm install
npm start
```

Puis ouvrir `http://localhost:4200`.
